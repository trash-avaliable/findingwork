战斗模块：

伤害类型：持续constant（敌人附加dot状态持续扣血），减速speed_down（使敌人移速降低），震颤defense_down（降低敌人防御力），爆炸explosion（伤害必定暴击），真伤truehurt（伤害无视防御力，不吃暴击），普通normal（无任何特殊效果）
持续伤害计算：敌人血量-（每层dot伤害-敌人防御力）* 段数，每段数字独立显示
减速公式：敌人移速-减速，if小于0则为0
震颤公式：敌人防御力* 防御力百分比/敌人防御力-减防
爆炸公式：敌人血量+防御力-基础伤害* 武器暴击伤害
真伤公式：敌人血量-基础伤害
基础伤害：子弹攻击力+武器附加攻击力+武器大招附加攻击力+角色攻击力
最终伤害：基础伤害* 武器暴击伤害，通过武器暴击率概率触发

敌人：蜘蛛，三体，等
武器初始8种，玩家可自定义武器子弹
武器和子弹分离，武器决定射程，目标人数，范围，附加攻击力，冷却……；子弹决定基础攻击力……
每种武器都有特定大招攻击方式，共用同一管能量条，普通攻击不消耗能量条
能量条消耗计算公式：角色能量条上限-（技能消耗+武器消耗）
武器每种有一专属大招，可通过语音或键盘点击

策略模块：
玩家每升一级都有一次三选一选择一次buff，可获得武器槽/武器/子弹/武器攻击力提升/子弹攻击力提升/攻击力提升/移速提升/防御力提升/金币/能量条上限提高/武器暴击率提高/武器暴击伤害提高
武器槽：解锁（修改值int）个武器槽，上限为10
武器：武器池里随机获得一把武器
子弹：子弹池里随机获得一类子弹
武器攻击力提升：武器附加攻击力* （1+修改数值(float)）+修改数值(int)
子弹攻击力提升：子弹攻击力* （1+修改数值(float)）+修改数值(int)
攻击力提升：角色攻击力* （1+修改数值(float)）+修改数值(int)
移速提升：移速+修改数值（int）
防御力提升：角色防御力* （1+修改数值(float)）+修改数值(int)
金币：角色持有金币*（1+修改数值(float)）+修改数值(int)
能量条上限：能量条上限+修改数值（int）
武器暴击率提高：武器暴击率*（1+修改数值(int)）+修改数值(float)
武器暴击伤害提高：武器暴击伤害*（1+修改数值(int)）+修改数值(float)

全局怪物数据管理单例EnemyDatabase：
# enemy_database.gd (AutoLoad)
extends Node

var enemies: Dictionary = {}      # key = id, value = EnemyData
var wave_config: Dictionary

func _ready():
    load_all_enemy_data()
    load_wave_config()

func load_all_enemy_data():
    var dir = DirAccess.open("res://data/enemies")
    if dir:
        dir.list_dir_begin()
        var file_name = dir.get_next()
        while file_name != "":
            if file_name.ends_with(".tres"):
                var path = "res://data/enemies/" + file_name
                var data = ResourceLoader.load(path) as EnemyData
                if data:
                    enemies[data.id] = data
            file_name = dir.get_next()

func load_wave_config():
    var file = FileAccess.open("res://config/waves.json", FileAccess.READ)
    var content = file.get_as_text()
    wave_config = JSON.parse_string(content)
